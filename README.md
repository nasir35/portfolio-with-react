# Bikemart

Bikemart is an eccomerce web app built on the MERN stack. Users can view products available for sale, see the description and prices, and can place the order. However certaing actions like accessing Dashboard, giving a review and placing orders is only allowed for logged in users. Authentication and Authorization use JSON web tokens which are generated in the backend and are provided when a user enters valid credentials in the login page.

- [Visit Live site](https://bikemart-client-side.vercel.app/)

- [Backend repository](https://bikemart-client-side.vercel.app/)

![Home](https://res.cloudinary.com/dax7yvopb/image/upload/v1720621698/bikemart-home-page_mrw8ha.png)

## Features And Tools

- Bikemart is a single page web app with a dashboard where people can buy bikes of different categories. Can see their order list and can manage them.
- The dashboard is designed differently for user and admin. An user can see his/her order list and can cancel order. And User can give review too!
- An admin can manage orders, products. He can add a Product or Blog and can make an user admin too.
- The dashboard is designed with Tailwind CSS and made functional with react-router-dom @6.0.
- The website made with React.js, Tailwind CSS, Firebase Authentication, Node.js, Express.js, Mongoose.
- Implemented JWT token to ensure the security of user/admin access.
- Backend design following MVC pattern and secured api ends using middlewares.
